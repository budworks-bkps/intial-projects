\# Workflow

Whenever a new design is uploaded:

1. Analyze the design carefully.

1. Compare it with the approved HTML template.

1. Reuse existing table structures wherever possible.

1. Never redesign the email.

1. Keep all responsive behavior unchanged.

1. Match every

- spacing
- typography
- colors
- paddings
- margins
- button size
- alignment

pixel perfectly.

1. Images should use placeholders.

Example

images/banner.png

images/icon1.png

images/card1.png

1. Generate production-ready HTML.

1. Validate Outlook compatibility.

1. Validate Gmail compatibility.

1. Validate iOS compatibility.

1. Validate Android compatibility.

1. Return only HTML.
